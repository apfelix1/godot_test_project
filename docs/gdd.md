# 游戏设计文档 (GDD)

## 概览

- 游戏名称：遗迹搬运队 / Ruin Crew
- 类型：3D简单卡通俯视角合作探险闯关 + 局外轻养成
- 目标平台：PC (Windows)
- 引擎：Godot 4.3
- 核心玩法一句话描述：组队探遗迹、搬宝并撤离
- 目标玩家：喜欢《Overcooked》式协作混乱乐趣，也喜欢轻养成、轻采集和欢乐派对体验的玩家
- 差异化卖点：把“下墓寻宝流程”抽象成轻奇幻遗迹探险，不做写实盗墓或硬核恐怖；核心乐趣不是纯战斗，而是分工探路、开机关、搬运宝物和安全撤离，局外再用营地、装备、技能的轻成长承接长期体验
